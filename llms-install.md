# Installation Guide for AI Agents

## Prerequisites
- Rust (latest stable)
- Cargo
- Git

## Installation Steps
1. Clone the repository: `git clone https://github.com/Scottcjn/rustchain-mcp.git`
2. Navigate to the directory: `cd rustchain-mcp`
3. Install dependencies: `cargo build --release`
4. Run the server: `cargo run`

## Configuration
- Set environment variables in `.env` file
- Configure RPC endpoints for RustChain blockchain
- Set up wallet credentials for BoTTube integration

## Usage
The server provides MCP tools for:
- RustChain blockchain interactions
- BoTTube video platform integration
- AI agent tools for earning RTC tokens
